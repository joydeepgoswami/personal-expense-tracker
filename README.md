# Personal Expense Tracker (Python)

## Description
This is a simple Python-based personal expense tracker.
It allows users to record daily expenses, store them in a CSV file,
and track monthly spending against a fixed budget.

## Features
- Add daily expenses (date, category, amount, description)
- Store expense data in a CSV file
- View all recorded expenses
- Track total expenses and remaining budget
- Update budget if expenses exceed the limit

## Tools Used
- Python
- Pandas
- CSV file handling
- Datetime module

## How to Run
1. Make sure Python is installed on your system
2. Install required library:
   pip install pandas
3. Run the program:
   python expense_tracker.py

## Project Type
Beginner-level Command Line Interface (CLI) project.
